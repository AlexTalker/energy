Building an ISO

Requirements

  • Arch Linux running on x86_64
  • archiso

Instructions

  1. Clone it.
  2. Run make.
  3. Done.

Installing

  1. Write an ISO to CD or USB stick.
  2. Boot it up.
  3. Run ./install.sh.
